FlutterSound class

This is THE main Flutter Sound class. This class is not instanciable. Use the expression FlutterSound() when you want to get the Singleton.

This class is used to access the main functionalities of Flutter Sound. It declares also a default FlutterSoundPlayer and a default FlutterSoundRecorder that can be used by the App, without having to build such objects themselves.
